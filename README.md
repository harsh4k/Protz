# Portfolio v1 (Archived)

First iteration of my personal portfolio. Superseded by [portfolio-v2](https://github.com/harsh4k/portfolio-v2), kept for reference.

## Tech Stack
- React 19 + Vite 6
- Express (backend server)
- Google Gemini API (`@google/genai`)
- Tailwind CSS 4
- Motion + Lenis

## Run Locally
```powershell
npm install
npm run dev
```
Requires a `.env` file — see `.env.example` for required keys (Gemini API key).
